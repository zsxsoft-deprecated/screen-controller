`data_id` (int, auto-increment, unique)
	no extend data

`data_sort` (int)
	use for sort

`data_player` (text, json) 
	use for store the player's info.
	{
		"name": "player_name",
		"class": player_class,
		"doom": "player_doom",
	// and other info
	}


`data_player_name` (text)
	the same as `data_player`.name
	use for search quickly


`data_player_class` (text)
	the same as `data_player`.class
	use for search quickly


`data_bgm` (text)
	use for store the URL of the background-music.
	If empty, that means no bgm.


`data_program` (text, json)
	use for store the program's info.
	{
		"name": "program_name",
		//etc..
	}


`data_program_name`(text)
	the same as `data_program`.name
	use for search quickly


`data_score`(text, array with int)
	use for store the score of the program
	[SCORE1, SCORE2, SCORE3, etc..]


`data_display`(text, array with json)
	use for store the objects that the screen have to display
	[
		{
			"message": "MESSAGE_DISPLAY_IN_SCREEN",
			"media": "MULTIMEDIA_DATA_DISPLAY_IN_SCREEN",
			"type": "MULTIMEDIA_TYPE(image|video|text)",
		}
		//etc..
	]
