screen-controller

LED屏幕控制器

Download this project as a .zip file Download this project as a tar.gz file

程序特性

  1. HTML5 + Node，模块化设计，可移植，定制性强，跨平台。
  2. 背景与音频统一管理。
  3. 一个节目允许多个背景和多曲背景音乐。
  4. 提供控制面板，允许对播出的节目和音乐进行切换。
  5. 允许通过无线网络等使用手机控制大屏幕。
  6. 允许使用视频作为背景。
  7. 提供打分并显示到屏幕功能。
  8. 提供弹幕功能。

制作原因

如果你不是负责场控的相关人员，你可能不会知道这些苦恼。

市面上常见的LED控制软件，如“LED演播室”，只能用于播放PowerPoint文件和固定的几张截图，音频需要使用其他播放器来播放。更恶心的是，操作的时候必须要调出面板或者设置定时，其他毫无办法。

作为福州八中三江口校区的第一届学生会信息部的部长，我对此深恶痛绝。怎么处理？那就自己造轮子！这套系统经过了两年来（2013年开始）的各种摧残，由事实证明了它的稳定性和通用性。它是由我在使用其他软件对场面进行控制时总结出来的经验而凝聚成的结果。
